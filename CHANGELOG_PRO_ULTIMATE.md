# Smart Farm PRO Ultimate Firmware - Changelog

โปรแกรมนี้ได้รับการอัปเกรดจากเวอร์ชันพื้นฐานเป็นเวอร์ชัน **PRO Ultimate** เพื่อความเสถียรและความปลอดภัยสูงสุด โดยมีการเปลี่ยนแปลงหลักดังนี้:

## 1. การเชื่อมต่อ MQTT (Security & Stability)
- **TLS/SSL Encryption:** เปลี่ยนจาก `WiFiClient` เป็น `WiFiClientSecure` เพื่อเชื่อมต่อกับ HiveMQ Cloud ผ่านพอร์ต **8883** อย่างปลอดภัย
- **Unique Client ID:** ใช้ `ESP.getChipId()` เพื่อสร้าง ID ที่ไม่ซ้ำกัน ป้องกันปัญหาการเชื่อมต่อซ้ำซ้อน
- **Auto Reconnect:** ระบบตรวจสอบการเชื่อมต่อ WiFi และ MQTT ตลอดเวลา หากหลุดจะทำการ Reconnect อัตโนมัติโดยไม่ทำให้โปรแกรมหยุดทำงาน (Non-blocking)

## 2. ระบบควบคุมหลายโซน (Multi-Zone Control)
- **4 Relays Support:** รองรับการควบคุมปั๊มหลัก และโซนย่อยอีก 3 โซน (Zone 1, 2, 3)
- **Command Structure:** รองรับคำสั่ง MQTT แบบเจาะจงโซน เช่น `smartfarm/cmd/zone1/on`
- **Persistence:** ใช้ **EEPROM** ในการบันทึกสถานะล่าสุดของรีเลย์ หากไฟดับหรือรีสตาร์ท ระบบจะกลับมาทำงานในสถานะเดิม

## 3. เซ็นเซอร์และการจัดการข้อมูล (Advanced Sensing)
- **Safe Reading:** ตรวจสอบค่า `NaN` จากเซ็นเซอร์ DHT และมีระบบ Retry หากอ่านค่าไม่ได้
- **RTC Integration:** รองรับ **DS3231 RTC** เพื่อการจัดการเวลาที่แม่นยำ (Real-time)
- **JSON Format:** ส่งข้อมูลเซ็นเซอร์ในรูปแบบ JSON เพื่อให้ Dashboard นำไปใช้งานได้ง่าย

## 4. ระบบอัตโนมัติ (Smart Automation)
- **Soil Threshold:** ปั๊มน้ำจะทำงานอัตโนมัติหากความชื้นในดินต่ำกว่าค่าที่กำหนด
- **Safety Timer:** มีระบบนับเวลาถอยหลัง (Duration) เพื่อปิดปั๊มอัตโนมัติ ป้องกันน้ำล้นหรือปั๊มไหม้
- **Manual Override:** สามารถสั่งเปิด-ปิดด้วยมือได้ตลอดเวลา แม้อยู่ในโหมดอัตโนมัติ

## 5. การแจ้งเตือนและการจัดการ (Monitoring & Management)
- **Telegram Bot:** แจ้งเตือนสถานะสำคัญ (WiFi, MQTT, Pump ON/OFF) ไปยังมือถือทันที
- **Dashboard API:** มี HTTP API (JSON) สำหรับเรียกดูสถานะผ่านเบราว์เซอร์หรือแอปอื่นๆ
- **OTA Update:** รองรับการอัปเกรดโปรแกรมผ่าน WiFi โดยไม่ต้องถอดบอร์ดมาเสียบสาย
- **Watchdog:** ระบบป้องกันโปรแกรมค้าง (WDT) จะรีสตาร์ทตัวเองหากพบความผิดปกติ

## 6. คุณภาพโค้ด (Code Quality)
- **Non-blocking Loop:** ใช้ไลบรารี `Ticker` แทน `delay()` เพื่อให้ระบบตอบสนองได้รวดเร็วตลอดเวลา
- **Modular Code:** แยกส่วนการทำงานเป็นฟังก์ชันชัดเจน ง่ายต่อการแก้ไขและบำรุงรักษา
- **Memory Optimization:** จัดการหน่วยความจำอย่างมีประสิทธิภาพเพื่อป้องกัน Memory Leak

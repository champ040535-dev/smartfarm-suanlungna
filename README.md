# 🌿 คู่มือระบบ Smart Farm (สวนลุงนะ)

ระบบนี้ประกอบด้วย 2 ส่วนหลัก คือ **Dashboard (HTML/JS)** สำหรับควบคุมผ่านเว็บ และ **NodeMCU/ESP8266 Firmware (.ino)** สำหรับติดตั้งที่อุปกรณ์ในฟาร์ม

## 1. การปรับปรุงระบบ Dashboard
ผมได้ทำการปรับปรุงไฟล์ `index_updated.html` ให้มีความเสถียรและรองรับการใช้งานจริง ดังนี้:
- **พิกัดพยากรณ์อากาศ:** อัปเดตเป็น ต.ปันแต อ.ควนขนุน จ.พัทลุง (7.7333, 100.0000)
- **การรับข้อมูล:** ปรับให้รองรับข้อมูลรูปแบบ JSON จากเซ็นเซอร์จริง (ESP32)
- **ความเสถียร:** เพิ่มระบบตรวจสอบสถานะ Online/Offline ของอุปกรณ์
- **กราฟ:** แสดงผลข้อมูลอุณหภูมิและความชื้นดินแบบ Real-time

## 2. การใช้งานไฟล์ .ino (ESP32)
ไฟล์ `smart_farm_esp32.ino` ถูกเขียนขึ้นเพื่อให้ทำงานร่วมกับ Dashboard ได้ทันที:
- **MQTT Broker:** เชื่อมต่อกับ HiveMQ Cloud (ตามค่าเดิมในไฟล์ HTML)
- **Topic ที่ใช้:**
    - `farm/sensor/data`: ส่งค่า Temp, Hum, Soil, Light (รูปแบบ JSON)
    - `farm/pump/control`: รับคำสั่ง ON/OFF จากเว็บ
    - `farm/pump/status`: ส่งสถานะปั๊มกลับไปที่เว็บ
- **การตั้งค่า:**
    - แก้ไข `ssid` และ `password` ให้เป็นของ WiFi ที่บ้านท่าน
    - ตรวจสอบขา Pin (GPIO) ให้ตรงกับที่ท่านต่อวงจรจริง

## 3. รายการไฟล์ที่ส่งมอบ
| ชื่อไฟล์ | คำอธิบาย |
| :--- | :--- |
| `index_updated.html` | ไฟล์เว็บ Dashboard ที่ปรับปรุงแล้ว |
| `smart_farm_esp8266.ino` | ไฟล์โปรแกรมสำหรับบอร์ด NodeMCU/ESP8266 |
